# OpenCV Computer Vision Projects

## Overview

This repository contains my learning journey in OpenCV, MediaPipe, and Computer Vision.

It includes image processing fundamentals, object detection, color tracking, transformations, thresholding techniques, hand tracking, and machine learning based gesture recognition experiments.

## Topics Covered

### Image Processing

- Thresholding
- Adaptive Thresholding
- Image Smoothing
- Image Pyramids
- Edge Detection
- Template Matching

### Image Transformations

- Affine Transformation
- Perspective Transformation
- Geometric Transformations

### Object Detection & Tracking

- Color Detection
- Object Position Detection
- Finger Detection
- Virtual Drawing

### MediaPipe Projects

- Hand Tracking
- Gesture Recognition
- Dataset Collection
- Model Training

### Machine Learning

- Gesture Dataset Training
- Gesture Classification
- Trained Model Deployment

## Technologies Used
